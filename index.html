Introduction:

This game has been a crowd favorite amongst my friends for a long time. I always wanted to be able to play with more than 10 players though so I built this website as a platform to customize Secret Hitler. This game version also includes a way to vote for your presidency anonymosly to give the facists the extra secrecy and increase the level of mystery in the game. So gather your friends, start a game and find Hitler!

How It Works:

One player in the group will select "Create Game." This will produce a random 4 letter room code. Each additional player will select "Join Game" and enter the room code. The game will ask each player for a name and then let them join the room. Once the group leader selects "Play!" The roles will be randomly sent out and no additional players may be added. The game will run all the voting and card selection.
